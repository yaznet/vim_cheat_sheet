Vim c’est comme Vi mais en version améliorée.

Vim fournit un programme d’apprentissage pour vous familiariser avec les commandes les plus courantes.
Pour le lancer il vous suffit d'aller dans un terminal et de taper : vimtutor

Vim a trois modes de fonctionnement :

  • Mode interactif : c'est le mode par défaut par lequel vous commencez. En lançant Vim, vous êtes donc en mode interactif.
    Dans ce mode, vous ne pouvez pas écrire de texte.)

  • Mode insertion : ce mode vous permez de tapez du texte
    Pour entrer dans ce mode, appuyer sur la touche i (insertion). (pour en sortir, il faut appuyer sur la touche Echap.)

  • Mode commande : ce mode permet de lancer des commandes telles que « quitter », « enregistrer ». Vous pouvez aussi l'utiliser pour activer des options de Vim (comme la coloration syntaxique, l'affichage du numéro des lignes...).
    Pour activer ce mode, vous devez être en mode interactif et appuyer sur la touche deux points « : ».
