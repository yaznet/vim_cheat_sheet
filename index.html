Cheat Sheet Vim

Commandes de base - Commandes d'édition - Recherche/remplacement - Fenêtrage

Vim c’est comme Vi mais en version améliorée.

Vim fournit un programme d’apprentissage pour vous familiariser avec les commandes les plus courantes.
Pour le lancer il vous suffit d'aller dans un terminal et de taper : vimtutor

Vim a trois modes de fonctionnement :

  • Mode interactif : c'est le mode par défaut par lequel vous commencez. En lançant Vim, vous êtes donc en mode interactif.
    Dans ce mode, vous ne pouvez pas écrire de texte.)

  • Mode insertion : ce mode vous permez de tapez du texte
    Pour entrer dans ce mode, appuyer sur la touche i (insertion). (pour en sortir, il faut appuyer sur la touche Echap.)

  • Mode commande : ce mode permet de lancer des commandes telles que « quitter », « enregistrer ». Vous pouvez aussi l'utiliser pour activer des options de Vim (comme la coloration syntaxique, l'affichage du numéro des lignes...).
    Pour activer ce mode, vous devez être en mode interactif et appuyer sur la touche deux points « : ».

Commandes de base :

  • h - déplacer le curseur vers la gauche
  • j - déplacer le curseur vers le bas
  • k - déplacer le curseur vers le haut
  • l - déplacer le curseur vers la droite
  • i - Passer dans le mode insertion
  • A - Ajouter en fin de ligne
  • :q - Quitter
  • :q! - Quitter sans enregistrer
  • :w - Enregistrer le fichier
  • :wq - Enregistrer et quitter
  • :x - Enregistrer (en cas de modification) et quitter

Commandes d'édition :

  • u - Annuler la dernière opération
  • <control>-r - Rétablir la dernière opération annulée
  • . - Répéter la dernière opération d'édition
  • yy - Copier la ligne (4yy = 4 lignes)
  • p - Coller après (P = insérer avant)
  • x - Effacer le caractère
  • dw - Effacer le texte jusqu'à la fin du mot
  • diw - Effacer le mot sous le curseur

Recherche / remplacement :

  • / - Rechercher du texte
  • n - Rechercher l'occurence suivante
  • N - Rechercher l'occurence précédente
  • cw - Remplacer le texte jusqu'à la fin du mot
  • ciw - Remplacer le mot
  • C - Remplacer jusqu'en fin de ligne
  • . - Répéter la dernière opération d'édition
  • :%s/A/B/g - Remplacer tous (g) les A par des B

Fenêtrage :

  • <control-w>-s - Diviser horizontalement
  • <control-w>-v - Diviser verticalement
  • <control-w>-w - Passer à la fenêtre suivante
  • <control-w>-n - Ouvrir un fichier vierge dans une nouvelle fenêtre
  • <control-w>-
  • : tabnew nom_fichier - Ouvrir le fichier nom_fichier dans un onglet
  • <control-w>--q - Fermer la fenêtre
